# "Проект interactive-cube(Интерактивный куб)"

Стек: HTML, CSS, JS, Vite

# Структура проекта:
- src/ — исходные файлы проекта

# Важные файлы:
- index.html — HTML-файл главной страницы
- main.js — точка входа приложения
- src/styles — корневой файл стилей


## Стек технологий
- **HTML5** — разметка интерфейса
- **CSS3** (включая 3D‑трансформации, анимации, переменные) — визуальное оформление и эффекты
- **JavaScript** — интерактивность и управление состоянием
- **Vite** — сборка и разработка


## Особенности проекта
- Плавное 3D‑вращение куба при наведении
- Изменение цвета граней
- Эффекты тени и свечения
- Адаптивное поведение 


Запустите сервер разработки:

```
bash
npm run dev
```
